# Streamlit Tutorials

## Install & Import

```bash
pip install streamlit
```
```bash
import streamlit as st
```

## Run in CMD prompt or power shell 
```bash
streamlit run first_app.py
```

## Command Line
```bash
streamlit --help
streamlit run your_script.py
streamlit hello
streamlit config show
streamlit cache clear
streamlit docs
streamlit --version
```
